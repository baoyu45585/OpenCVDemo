# OpenCVDemo
OpenCV人脸捕捉和识别和灰度

人脸检测是一个非常经典的问题，但是还是有一些常见的问题出现在实际使用当中： 

（1）误检（把非人脸的物体当作人脸）较多，非人脸图像当作人脸送入后续算法，会引起一系列不良后果。 

（2）漏检问题，例如戴墨镜、大胡子、逆光条件、黑种人、倾斜姿态较大的脸无法检测到。
